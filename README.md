# Olist EC事業 売り上げ改善分析プロジェクト

ブラジル最大級のECモール Olistのデータを用い、売上成長のボトルネックを特定。

SQL（BigQuery）による現状分析から、Python（LightGBM）を用いた要因検証、施策のROIシミュレーションまでを一気通貫で実施しました。

## 📒プロジェクト概要
売上の伸び悩みを課題とし、売上を「購入ユーザー数 × 平均単価 × 平均購入回数」に分解。

分析の結果、リピート率（平均購入回数 1.033回）に最大のボトルネックがあることを特定。

LTV向上に向けた具体的なリピート改善施策を提案しました。

## 📊分析結果サマリー

### データの前処理

order_status が delivered（配送完了）以外のデータを除外。(キャンセルや配送途中のorderを除外)

顧客特定には customer_id ではなく customer_unique_id を使用し、リピート判定の精度を担保。(Olistでは customer_id はorderごとに発行されるため)

### 現状把握：リピート率の課題
**ボトルネック: 全体のリピート率は約3%。売上高の約94%が単発購入ユーザーに依存しており、LTVがほぼ成立していない。

**再購入分布：初回購入から1〜2日以内の追加購入を除くと、30日以降の再購入が極端に低い。

**コホート分析: 2016年から2018年にかけ、継続率は4% → 1%へと低下傾向にあり、早期離脱が顕著。

### セグメント・要因分析
**単価別: 高単価（1,001~）ほどリピート率が低い（1.57%）傾向。

**カテゴリ別: cama_mesa_banho（寝具等）やbeleza_saude（美容・健康）がユーザー数・リピート率共に改善インパクトが大きい。

**機械学習による検証: LightGBMを用いたFeature Importance（特徴量重要度）を算出した結果、単純な相関分析では見えなかった「配送日数」と「購入単価」がリピートを左右する支配的な要因であることを特定。

## 💡提案施策：リピート率改善アプローチ

### ターゲット：cama_mesa_banho（寝具等）やbeleza_saude（美容・健康）(インパクト◎ = ユーザー数大　×　リピート率中)
初回購入から1ヶ月間の離脱を防ぐため、以下のステップで施策を実行。

**購入直後: 関連商品のレコメンド表示による追加購入の促進。

**1〜3週間後: 週次の需要（週末利用等）に合わせたクーポン配布。

**1ヶ月後: 最後のリマインドメール配信（過度な配信による悪評防止のためこれ以降は停止）。

## 👓改善予測効果と検証
**経済インパクトの試算
　平均購入回数を 1.033回 → 1.1回 へ引き上げることで、2年間で約3,202万円以上の増収を予測。

**統計的有意性の検証（A/Bテストシミュレーション）
　仮想のA/Bテストにおいて、クーポン配布によるリピート率の改善幅（+1.1%）が有意であるかを2標本比率検定で確認。
 
 結果: z 値 = 3.7 , p値 = 0.0002
 
 判断: 有意水準1%でも統計的に有意であり、1人あたり純増益 +700円が見込めるため、本施策は施行価値があると判断。
